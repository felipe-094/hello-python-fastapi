# 🚀 Hello Python FastAPI

Proyecto desarrollado con Python y FastAPI para aprender el desarrollo de APIs modernas y aplicaciones backend.

## 🛠 Tecnologías

* Python
* FastAPI
* Git
* GitHub

## 📂 Contenido

* Backend
* Basic
* Intermediate

## 🚀 Instalación

Clonar el repositorio:

git clone https://github.com/felipe-094/hello-python-fastapi.git

Entrar al proyecto:

cd hello-python-fastapi

Instalar dependencias:

pip install -r requirements.txt

Ejecutar el servidor:

uvicorn main:app --reload

## 🎯 Objetivo

Este proyecto forma parte de mi aprendizaje en desarrollo backend utilizando FastAPI y Python.
